# Day 04 - Semantic HTML & Multimedia

## What is Semantic HTML?

Semantic HTML uses elements that describe the purpose of the content.

Examples:
- header
- nav
- main
- section
- article
- aside
- footer

Benefits:
- Better readability
- Better SEO
- Better accessibility
- Easier maintenance

---

## Semantic Tags

### `<header>`
Top section of a webpage.

### `<nav>`
Contains navigation links.

### `<main>`
Primary content of the page.

### `<section>`
Groups related content.

### `<article>`
Independent piece of content.

### `<aside>`
Related or supplementary content.

### `<footer>`
Bottom section of the webpage.
