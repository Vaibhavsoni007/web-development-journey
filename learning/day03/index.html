day03 - HTML Forms

Form Structure







Form Input Field Attributes

Placeholder: A placeholder is a short hint or description that appears inside an input field when it is empty. It provides guidance to the user about what type of information is expected in that field.

Type: The type attribute specifies the type of input field. It determines the kind of data that can be entered into the field, such as text, email, password, number, etc.

Value: The value attribute specifies the initial value of the input field. It can be used to pre-fill the field with a default value or to set a specific value that will be submitted with the form.

Required: The required attribute is a boolean attribute that indicates whether the input field must be filled out before submitting the form. If an input field has the required attribute, the user will not be able to submit the form without providing a value for that field.

Example of Input Types

Password: Used for entering passwords. The input is masked.

Email: Used for entering email addresses. Validates the format.

Number: Used for entering numeric values.

Date: Used for selecting a date.

Radio: Used for selecting one option from a group.

Checkbox: Used for selecting one or more options from a group.

File: Used for uploading files.

Submit: Used for submitting the form data to the server.

Reset: Used for resetting the form fields to their default values.
